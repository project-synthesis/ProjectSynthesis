# Project Synthesis — Project Vision

> The system that builds software also builds a better version of itself.

Every AI system runs on language. The instructions fed into agents, pipelines, and autonomous workflows determine their output far more than the models themselves — yet prompt engineering has no tooling infrastructure. No compiler catches logical ambiguity. No version control surfaces why a result degraded. No institutional memory captures what works for a specific codebase or domain. Teams rediscover the same failure modes independently, in silence.

Project Synthesis is that missing infrastructure. It runs prompts through a structured five-stage pipeline — reading the actual codebase for context, classifying weaknesses, selecting an optimization framework, rewriting with precision, then scoring across multiple dimensions with actionable feedback. Every run is stored. Every score is tracked. Improvement becomes measurable, not assumed.

The deeper ambition is a platform that learns from its own operation. Optimized prompts drive an autonomous multi-agent coding system that builds and tests full-stack applications. After every run, the system analyzes its own performance, generates improvements to its strategy library, and applies them automatically. Each cycle compounds on the last. The ceiling keeps rising because the system is always raising it.

Project Synthesis is free and open source software, designed to run entirely on your own infrastructure. Your prompts, codebase context, and optimization history stay local. The community improves the shared codebase — new frameworks, better rubrics, sharper strategy selectors — and every self-hosted installation benefits through the normal update cycle.
